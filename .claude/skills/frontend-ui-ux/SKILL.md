## Skill Overview

A designer-turned-developer Skill that blends visual craft with production-grade front-end engineering. It creates pixel-perfect layouts, refined spacing, color harmony, and memorable micro-interactions using HTML/CSS/JS and modern frameworks (React, Vue, Angular). Key features include rapid aesthetic direction (purpose, tone, constraints, differentiation), implementation of accessible, performant components and animations, creation or refinement of design systems and tokens, and seamless integration with existing code patterns. Use cases: building new UIs from concept, converting design-less specs into cohesive interfaces, elevating component libraries, fixing visual regressions, and performing design-aware code reviews. Core advantages are fast, high-fidelity UI delivery, maintainable code that matches team conventions, improved UX cohesion, and clear, transparent documentation and verification throughout the development process.

## Search Tips

Use task-oriented keywords. Instead of generic terms, describe what you need to do:

- Need to edit images → `--q "image editor"`
- Need to work with Excel files → `--q "excel spreadsheet"`
- Need to send emails → `--q "email smtp"`
- Use `--sort installCount` if you want to sort by popularity explicitly

## Examples

