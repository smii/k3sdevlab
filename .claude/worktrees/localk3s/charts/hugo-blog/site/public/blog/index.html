Hoe het allemaal begon: DOS installeren als 8-jarige 

Ik kom uit een familie die eind jaren 80 al in de IT zat en kreeg mijn eerste 8086 (JA een 8086) met 4 MB schijf, 512 KB RAM en een zwart-zwit scherm toen ik 8 jaar oud was. 

Bij deze pc kreeg ik de instructie dat ik enkel bestanden kon starten die eindigde met .exe in de directory ‘games’, maar hiervoor was ik te eigenwijs. Ik vond andere directories ook interessant en speelde met programmas uit de andere directories, zoals de DOS directory. Ook startte ik fdisk ( link voor de jonge lezers ) op, met als gevolg dat DOS niet meer startte en ik ook niet wist hoe ik dat kon repareren. Ja, ik dacht dat mijn computer kapot was.
